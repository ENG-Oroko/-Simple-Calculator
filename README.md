# 🧮 Modern Web Calculator

A simple, modern calculator built using **HTML, CSS, and JavaScript**.  
It supports basic arithmetic operations, keyboard input, and a calculation history system.

---

## 🚀 Features

- ➕ Basic operations: +, −, ×, ÷, %
- ⌨️ Full keyboard support
- 📜 Calculation history panel
- 🖱️ Click history to reuse previous calculations
- 🧹 Clear history button
- 📱 Responsive design (works on mobile & desktop)
- 🎨 Modern UI with smooth styling and animations
